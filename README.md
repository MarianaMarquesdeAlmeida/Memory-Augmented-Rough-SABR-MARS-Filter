# Memory-Augmented Rough-SABR Particle Filter

Code for my MSc thesis comparing Normal SABR and memory-augmented Rough-SABR particle filters using Bitcoin option data.

## Project structure

- `src/normal_sabr_pf.py` — Normal SABR particle filter
- `src/rough_sabr_pf.py` — Rough-SABR particle filter
- `notebooks/model_comparison.ipynb` — model comparison and analysis
- `data/raw/` — raw Deribit option data
